# Bibliotecas para segurança de dados

Segurança da informação é o conjunto de práticas, tecnologias e políticas adotadas para proteger dados e informações contra acessos não autorizados, uso indevido, alterações ou destruição. Essa área busca garantir a confidencialidade, integridade e disponibilidade dos dados, garantindo que apenas as pessoas ou sistemas autorizados tenham acesso e que as informações permaneçam íntegras e precisas. A criptografia desempenha um papel fundamental na segurança da informação, oferecendo métodos para proteger dados sensíveis por meio de algoritmos de criptografia simétrica e assimétrica, que serão explorados ao longo deste documento.

# O que é função hash
Uma função hash é uma transformação matemática que mapeia dados de entrada (como uma sequência de bytes) para um valor de hash fixo (geralmente uma sequência hexadecimal). Ela é utilizada para verificar a integridade dos dados e criar identificadores únicos.
- Exemplo: O SHA-256 e o SHA-512 são funções de hash amplamente utilizadas.

# O que é o algoritmo SHA-256
SHA-256 significa: Secure Hash Algorithm.
- O SHA-256 é um algoritmo de hash criptográfico amplamente utilizado.
- Ele pertence à família de algoritmos SHA-2, que produz hashes de 256 bits.
- O objetivo principal do SHA-256 é gerar um valor hash único e fixo para uma entrada de dados.
- É usado para verificar a integridade dos dados e para garantir que eles não foram alterados.
- O SHA-256 é comumente usado em sistemas de segurança, autenticação, assinaturas digitais e em várias aplicações criptográficas.

# O que é o algoritmo AES-256
- O AES-256 é um algoritmo de criptografia simétrica amplamente adotado.
- Ele foi selecionado como o padrão de criptografia pelo NIST (National Institute of Standards and Technology) dos Estados Unidos.
- O AES-256 opera em blocos de 128 bits e suporta chaves de 256 bits.
Ele oferece um alto nível de segurança e é considerado resistente a ataques de força bruta.
- O AES-256 é usado para criptografar dados de forma segura, garantindo que somente as partes autorizadas possam acessá-los.
- É utilizado em uma variedade de aplicações, incluindo comunicações seguras, armazenamento de dados sensíveis e proteção de informações confidenciais.
